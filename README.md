### 📈 Objetivo

Ajudar a guardar repositórios úteis.
<br />

<hr>
 
### 📝 Como utilizar

Clique em Buscar para explorar diversos repositórios no github, os repositórios pesquisados ficam na tela inicial após serem buscados.
